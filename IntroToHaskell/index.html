Sin taxes

Things that do not do what you think they do:

  • . function compostion
  • ( ) think order of operations
  • : singleton list prepend (throw an element on the front of a list)
  • $ used to change the order of exectution
  • != equivalent is \=
  • -- actually a one line comment, like //
  • ++ not an incrementer (that wouldn't really make sense anyway) actually list join
  • % not used mod is what you want

Things that do what you'd expect:

  • Most math operations: + - * /
  • Most boolean operators: == && || > < <= >=

I need a filter

So suppose you have a list (or array) and you want to remove the elements that fail a condition.

in Java:

1 ArrayList<Integer> filter(ArrayList<Integer> list){
2     ArrayList<Integer> result = new ArrayList<Integer>();
3     for( int a : list)
4         if( test(a) == True)
5             result.append(a)
6     return result;
7 }

in Haskell:

1 filter :: (a -> Bool) -> [a] -> [a]
2 
3 filter test [] = []
4 filter test (x:xs)
5     | test x    = x : filter test xs
6     | otherwise = filter test xs

Lucas!

Find the number of even lucas numbers below 4,000,000 Lucas numbers follow the same rules as fibonacci numbers, but have the start values of 2 and 1, rather than 0 and 1

 1 -- calculate lucas numbers based on fibonacci numbers
 2 lucas :: Integer -> Integer
 3 lucas n = fib (n - 1) + fib (n + 1)
 4 
 5 -- Using Binet's formula to find fibonacci numbers
 6 -- golden ratio
 7 
 8 fib :: Integer -> Integer
 9 fib n = round $ phi ** fromIntegral n/sq5
10     where
11         sq5 = sqrt 5 :: Double
12         phi = (1+ sq5) / 2
13 main = print $ length $ filter (even) $ takeWhile (<4000000) $ map lucas [1,2 ..]
