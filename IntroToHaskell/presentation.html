I need a filter

So suppose you have a list (or array) and you want to remove the elements that fail a condition.

in Java:

1 public ArrayList<Integer> filter(ArrayList<Integer> list){
2     ArrayList<Integer> result = new ArrayList<Integer>();
3     for( int a : list)
4         if( test(a) == True)
5             result.append(a)
6     return result;
7 }

in Haskell:

1 filter :: (a -> Bool) -> [a] -> [a]
2 
3 filter test [] = []
4 filter test (x:xs)
5     | test x    = x : filter test xs
6     | otherwise = filter test xs

Presenter Notes

base case, x:xs, otherwise, guards, actual implementation of filter from Data.List

Folding, at home!

Find the sum of the numbers from 1 to some number n.

in Java:

1 public int factorial(int n){
2     int sum = 0;
3     for(int i=0; i <n; i++){
4         sum += i;
5         }
6     return sum;
7 }

in Python:

1 def factorial(n):
2     sum = 0
3     for i in range(1, n):
4         sum += n
5     return sum
